# 🌤️ Weather App

A simple and responsive Weather App built using **HTML, CSS, and JavaScript**. It allows users to search for any city and get real-time weather information using a weather API.


## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Displays current temperature
- 💧 Shows humidity
- 🌬️ Displays wind speed
- ☁️ Dynamic weather icons
- 📱 Responsive design
- ⚡ Fast and easy to use

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Open the project folder

```bash
cd weather-app
```

3. Open `index.html` in your browser.

---

## 🔑 API

This project uses a **temporary Weather API key** for learning purposes.

To use your own API key:

1. Create an account on your preferred weather API provider.
2. Generate an API key.
3. Replace the existing API key in `script.js`.

Example:

```javascript
const apiKey = "YOUR_API_KEY";
```
